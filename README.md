# Mini Projects

A collection of simple, hands-on mini projects built using Python for practice and learning.

These projects focus on strengthening core programming concepts, problem-solving, and basic automation through small, practical applications.

## Current Projects
- Calculator — Command-line calculator  
- File Organiser — Organises files by extension  
- PDF Merger — Merges multiple PDF files into one  
- QR Code Generator — Generates QR codes from URLs  

Each project includes its own README with usage instructions.

This repository will continue to grow as more projects are added over time.
