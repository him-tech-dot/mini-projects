# PDF Merger (Python)

A simple command-line Python program that merges multiple PDF files into one PDF file.

## What it does
This program asks the user for the number of PDF files and their names, then combines them into a single file called `merged-pdf.pdf`.

## Requirements
- Python 3.x
- PyPDF2 library

Install dependency:
pip install PyPDF2

## How to use

1. Place the PDF files you want to merge in the same folder as this script.

2. Run the program:
python main.py

3. Enter the number of PDF files and their file names when prompted.

4. The merged file will be created as:
merged-pdf.pdf

## Example

enter no.of pdf to be merged :
3
enter name of pdf 1: file1.pdf
enter name of pdf 2: file2.pdf
enter name of pdf 3: file3.pdf

Output:
merged-pdf.pdf is created in the same folder.
